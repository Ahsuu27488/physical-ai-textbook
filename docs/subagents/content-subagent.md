# Content Subagent

## Purpose
Orchestrates content delivery by coordinating between personalization and translation features.

## Responsibilities
- Manage content delivery workflows
- Coordinate between personalization and translation
- Handle user experience flows
- Integrate with frontend components

## Implementation
The ContentSubagent acts as the central coordinator that brings together authentication, personalization, and translation services to deliver customized content to users.

## Coordination
- Interfaces with AuthSubagent for authentication checks
- Works with PersonalizationSubagent for content customization
- Coordinates with TranslationSubagent for language conversion
- Communicates with frontend components for user interaction

## Agent Skills Used
- All other agent skills as needed for specific tasks
