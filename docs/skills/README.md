# Claude Code Agent Skills

This directory contains agent skills for the Physical AI & Humanoid Robotics Textbook Platform.

## Available Skills

1. [BetterAuth Integration](./betterauth-integration.md) - Handles user authentication with background collection
2. [Translation Service](./translation-service.md) - Manages content translation with caching
3. [Personalization Engine](./personalization-engine.md) - Applies content personalization based on user preferences
4. [Database Operations](./database-operations.md) - Handles all database operations

## Usage

These skills can be referenced by Claude Code agents to perform specific tasks related to authentication, translation, personalization, and database operations.

## Hackathon Requirements

These agent skills fulfill the requirement for "reusable intelligence via Claude Code Subagents and Agent Skills" for the hackathon.
